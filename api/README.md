# Todo System - API

## Rotas criadas na API

* Criar todo
```shell script
POST - /api/todos
```

* Listar todos todo
```shell script
GET - /api/todos
```

* Buscar uma todo pelo id
```shell script
GET - /api/todos/{id}
```

* Excluir um todo pelo id
```shell script
DELETE - /api/todos/{id}
```

```shell script
PATCH - /api/todos/{id}/done
```
